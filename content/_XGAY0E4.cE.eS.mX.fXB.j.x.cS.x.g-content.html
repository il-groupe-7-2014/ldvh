Logiciel de création de LDVH
Cas d'utilisation Sauvegarder un livre

Identification

Identifiant : 3

Nom du cas : "Sauvegarder un livre"

Description : L'auteur veut sauvegarder un livre qu'il a créé.

Acteur principal : Auteur

Date de création : le 13/10/2014

Date de mise à jour : le 13/10/2014

Responsable : Boris Berger

Version : 1.0

Séquencement

Pré-conditions :

Un livre doit etre ouvert dans le Logiciel.

Enchaînement nominal :

1. L'auteur clique sur le bouton "Sauvegarder Livre".

2. Le système affiche une boite de dialogue permettant de choisir un fichier.

3. L'auteur choisit un fichier "file".

4. Le système vérifie que le fichier "file" existe.

5. Le système affiche une confirmation d'écrasement de fichier.

6. L'auteur valide l'écrasement.

7. Le système écrit le contenu du livre ouvert dans le fichier ""file

Enchaînements alternatifs :"

A1 : Le fichier "file" n'existe pas

L'enchaînement démarre après le point SN4 et reprend en SN7

A1.1 Le système créer un fichier "file".

Enchaînements d'exception :

E1 : L'auteur annule la sauvegarde du livre.

L'enchaînement démarre après le point SN2 ou SN5 :

4. Le système affiche son interface usuelle (le graphe correspondant au livre).

Post-conditions :

Le livre a été sauvegardé par le système.
