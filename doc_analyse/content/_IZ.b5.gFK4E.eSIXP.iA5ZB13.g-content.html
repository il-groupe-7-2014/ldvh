Logiciel de création de LDVH
Cas d'utilisation Créer un enchaînement

Identification

Identifiant : 8

Nom du cas: "Créer un enchaînement"

But: décrire les étapes permettant à l'auteur de créer un enchaînement via le logiciel

Acteur: auteur

Date de création: 12/10/14

Date de modification: 12/10/14

Responsable: M. PEREZ

Version: 1.0

Séquencement

Le cas d'utilisation commence lorsque l'auteur clique sur le bouton "Créer un enchaînement".

Pré-conditions

Aucune.

Enchaînement nominal

1. L'auteur renseigne le nom de l'enchaînement et les prérequis dans les champs prévus à cet effet.

2. Le système affiche la confirmation.

3. L'auteur confirme la création de l'enchaînement.

Enchaînement alternatif

A1 : Un enchaînement ayant le même nom a déjà été créer.

L'enchaînement démarre après le point 1 de la séquence nominale :

2. Le système affiche dans une boîte de dialogue que le nom est déjà pris pour un enchaînement et invite l'auteur à modifier le champ prévu à cet effet.

Enchaînements d'exception

E1 : L'auteur annule l'opération

L'enchaînement démarre à n'importe quel point où l'auteur à la main.

Le système ferme la fenêtre de création d'enchaînement.

E2 : La création de l'enchaînement a échoué

L'enchaînement démarre après le point 3. de la séquence nominal.

4. Le système affiche un message indiquant à l'auteur que la création de l'enchaînement a échoué et lui propose d'en crer un autre.

Post-conditions

Le système a ajouter l'enchaînement créer par l'auteur par l'auteur dans l'interface graphique.
