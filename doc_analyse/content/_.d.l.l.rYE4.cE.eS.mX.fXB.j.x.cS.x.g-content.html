Logiciel de création de LDVH
Cas d'utilisation Générer un livre

Identification

Identifiant : 4

Nom du cas : "Générer un livre"

Description : L'auteur veut générer le livre qu'il a créé.

Acteur principal : Auteur

Date de création : le 06/10/2014

Date de mise à jour : le 06/10/2014

Responsable : Stéphane Meresse

Version : 1.0

Séquencement

Pré-conditions :

Le livre doit contenir au moins une section.

Enchaînement nominal :

1. L'auteur clique sur le bouton "Générer".

2. Le système lui propose deux choix : "Pages HTML" ou "Version imprimable".

3. L'auteur choisit : "Pages HTML".

4. Le système génère les pages HTML correspondantes au livre.

Enchaînements alternatifs :

A1 : Une version imprimable du livre est générée.

L'enchaînement démarre après le point 2 dans la séquence nominale :

3. L'auteur choisit : "Version imprimable".

4. Le système génère une version imprimable du livre.

Enchaînements d'exception :

E1 : Le livre contient une section inatteignable.

L'enchaînement démarre après le point 1 dans la séquence nominale :

2. Le système indique à l'utilisateur qu'il existe au moins une section inatteignable dans le livre et que celui-ci ne peut donc pas être généré.

3. L'auteur confirme.

4. Le système affiche son interface usuelle (le graphe correspondant au livre).

E2 : L'auteur annule la génération du livre.

L'enchaînement démarre après le point 2 dans la séquence nominale :

3. Le système affiche son interface usuelle (le graphe correspondant au livre).

Post-conditions :

Le livre a été généré par le système.
