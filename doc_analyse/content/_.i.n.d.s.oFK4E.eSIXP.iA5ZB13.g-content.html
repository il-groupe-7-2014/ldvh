Logiciel de création de LDVH
Cas d'utilisation Créer un objet

Identification

Identifiant : 11

Nom du cas : “Créer un objet”

But : création d’un objet lors de la conception d’un livre dont vous êtes le héros

Acteur principal : utilisateur

Acteur secondaire : néant

Date de création : le 12/10/2014

Date de mise à jour : le 12/10/2014

Responsable : Elliott Muenier

Version 1.1

Séquencement

Le cas d’utilisation commence lorsque l’utilisateur appuie sur le bouton de la fonctionnalité.

Pré-conditions

l’utilisateur doit avoir préalablement créé ou chargé un livre

Enchaînement nominal

1. l’utilisateur appuie sur le bouton de la fonctionnalité

2. le système affiche la fenêtre de création d’objet

3. l’utilisateur entre le nom d’un objet

4. le système vérifie qu’un des objets du livre (s’il en existe déjà) ne porte pas le même nom

7. l’utilisateur quitte l’assistant de création d’objet

Enchaînement alternatifs

A1 : le système détecte que le nom de l’objet est le même qu’un autre existant

L’enchaînement démarre après le point 4 de la séquence nominale :

4. Le système indique à l’utilisateur que le nom existe déjà et qu’il doit le modifier

La séquence nominale reprend à l’étape 3.

Post-conditions :

Le système a créé un nouvel objet.
